# Todolist README
## 1. 项目介绍
```
React Learn，只是一个简单的响应式『React Hooks事件待办』
```
## 1. 项目结构
```
src
├── component: 组件
│   ├── AddInput: 添加输入框
│   ├── TodoItem: 待办事项
│   └── Header: 标题栏
└── App.js 入口文件
```
## 2. 项目运行
    ```
    yarn install
    yarn start
    ```
## 3. 使用方法
切换到手机模式效果更佳
```
1. 输入待办事项
2. 点击添加
3. 点击待办事项完成
4. 点击待办事项删除
```
